# Coffee Sales Dashboard

## Project Overview

The Coffee Sales Dashboard provides an interactive view of coffee sales performance across different countries, customers, and product categories. It is designed to help users explore sales trends and gain actionable business insights through dynamic visualizations.

## Dataset used
- <a href="https://github.com/mochen862/excel-project-coffee-sales">Dataset</a>

## Key Features

* 📈 Sales trend analysis over time
* 🌍 Sales performance by country
* 👥 Top 5 customers by sales
* 🎛️ Interactive slicers for:

  * Roast Type
  * Coffee Size
  * Loyalty Card Status
  * Order Date
 
    <a href="https://github.com/Delight142/Data-Analysis-Dashboard/blob/main/Capture1.PNG">View Dashboard</a>

## Business Impact

This dashboard enables businesses to:

* Monitor sales performance over time
* Identify top-performing customers and markets
* Understand customer purchasing preferences
* Support data-driven sales and marketing decisions through interactive reporting
